## Purpose

CSS Cleaner is an extension for Visual Studio Code that automatically optimizes CSS code in your project.

Press: ctrl + shift + P

- Support for React components (`*.jsx`, `*.tsx`)
- Support for Angular components (`*.component.ts`)
